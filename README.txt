Lonz Shots Netlify-ready site

Files included:
- index.html
- styles.css
- script.js
- netlify.toml

Deploy on Netlify:
1. Log in to Netlify.
2. Drag the full folder contents into a new manual deploy, or upload the ZIP.
3. Netlify Forms will detect the "booking" and "subscribe" forms after first deploy.
4. If you want your own domain, add it in Netlify Domain settings.

Notes:
- The redesign keeps the original Lonz Shots content structure, but presents it in a cleaner premium layout.
- Images and logo currently load from the existing Wix static image URLs so the site can deploy immediately.
- You can later replace those image URLs with local files if you want the site fully independent from Wix.
